## Install

1. Install MSCLoader for My Winter Car
2. Put the dll & assets in your Mods folder

## Changelog

### 1.0.2

- Added grip adjustment

- Added custom livery/texture support (bachglotz_paint.png, bachglotz_paint_s.png)

- Initial release

- Custom license plate with Finnish format validation
