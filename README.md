## Install

1. Install MSCLoader for My Winter Car
2. Put the dll & assets in your Mods folder

## Features
- Second Bachglotz spawns near the player house
- Custom license plate with Finnish format validation (settings menu)
- Fuel level and position saved between sessions
- Reset button in mod settings if you flip it or lose it somewhere
- Custom paint color system with in-game RGB picker (Ctrl+P)
- Custom livery support (replace bachglotz_paint.png in Assets folder)
- Reset to original livery if you dont want to modify
- Heater warms up player
- Switch between RWD,FWD,AWD (mod settings)
- Front & Rear grip adjustment (mod settings)
- Tachometer toggle
